主术语与别名

主术语常见别名说明
Warehouse仓库仓储空间的总实体
Zone库区 / 区域仓库内的功能分区
Location库位 / 储位 / Bin库存实际存放位置
SKU物料 / 商品 / 库存单位最小管理粒度的商品
Inventory库存 / 库存明细 / 库存余额需要在系统中统一口径
Inbound Order入库单 / 收货单前置单据业务请求，不等于收货结果
Outbound Order出库单 / 发货单前置单据业务请求，不等于发运结果
Receipt收货单 / 收货结果入库执行结果
Shipment发运单 / 发货结果出库执行结果
Carrier承运商 / 物流商 / 运输商负责运输执行的主体
Putaway上架将库存放入目标库位
Picking拣货 / 取货按订单从库位取货
Packing包装 / 打包拣货后整理包装
Loading装车货物装上运输工具
Replenishment补货 / 补位将库存补到拣货位置
Cycle Count盘点周期性核对账实一致性

使用原则

  • 对外文档优先使用主术语
  • 系统字段可保留别名，但要写清对应关系
  • 当别名有歧义时，以本体定义为准
  • 不要让别名替代正式概念定义
