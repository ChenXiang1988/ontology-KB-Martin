# 术语表

| 术语 | 含义 |
| --- | --- |
| Ontology | 某领域知识的形式化表示 |
| RDF | 用三元组表达资源与关系的标准 |
| OWL | 表达更强逻辑约束的本体语言 |
| SKOS | 适合术语表和分类体系的标准 |
| Class | 概念类别 |
| Instance | 具体个体 |
| Property | 属性 |
| Relation | 关系 |
| Axiom | 约束或逻辑公理 |
| 3PL | 第三方物流 |
| WMS | 仓储管理系统 |
| SKU | 库存单位 |
| Lot | 批次 |
| SerialNumber | 序列号 |
| Location | 库位 |
| Zone | 库区 |
