# 核心概念

## 1. Class

表示一类概念，例如：

- `Person`
- `Organization`
- `Product`

## 2. Instance

表示具体个体，例如：

- `Alice`
- `OpenAI`
- `SKU-001`

## 3. Property

表示属性，例如：

- `name`
- `age`
- `foundedAt`

## 4. Relation

表示对象之间的关系，例如：

- `worksFor`
- `contains`
- `shipsTo`

## 5. Axiom

表示约束或逻辑规则，例如：

- `Manager` 一定是 `Employee`
- 冷链商品只能存放在冷库库位

## 6. Hierarchy

表示分类层次，例如：

- `Vehicle`
  - `Truck`
  - `Forklift`
