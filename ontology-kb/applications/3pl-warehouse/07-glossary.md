# 术语表

| 术语 | 含义 |
| --- | --- |
| Inbound Order | 入库单，客户发货到仓前的业务请求 |
| Outbound Order | 出库单，客户要求仓库发货的业务请求 |
| Receipt | 收货结果或收货单据 |
| Shipment | 发运结果或发运单据 |
| Putaway | 上架，把货放到指定库位 |
| Picking | 拣货，按订单从库位取货 |
| Packing | 包装，拣货后整理打包 |
| Loading | 装车，货物装上运输工具 |
| Replenishment | 补货，把库存补到拣货位 |
| Cycle Count | 盘点，按周期检查库存准确性 |
| SKU | 库存单位，最小管理商品粒度 |
| Lot | 批次，同一批生产或入库的货 |
| Serial Number | 序列号，单件唯一识别码 |
| Pallet | 托盘，承载货物的物流单元 |
| Container | 周转箱或容器 |
| Zone | 库区，仓库内部的功能区域 |
| Location | 库位，库存实际存放位置 |
| Available | 可用库存状态 |
| Reserved | 已预留库存状态 |
| Damaged | 损坏库存状态 |
| Quarantine | 隔离库存状态 |
| Wave | 波次，一组集中处理的出库任务 |
