这是一个面向 3PL 仓储领域的 ontology 应用案例包，用于说明如何把 ontology 方法应用到仓储业务建模、规则校验和知识组织中。

## 这套文档适合什么用途

- 作为 3PL 仓储 ontology 的案例底稿
- 作为 WMS 语义层建模参考
- 作为业务术语统一与规则梳理材料
- 作为后续落到 RDF / OWL / 知识图谱的前置文档

## 阅读顺序

- 先看背景与范围，确认边界
- 再看概念模型和结构图，理解对象与关系
- 再看规则、验证案例和系统映射，确认可落地性
- 最后看术语表和开放问题，补齐定义和未决事项
- 需要对外协作时，再看版本记录和术语别名表
