# 术语别名表

## 主术语与别名

| 主术语 | 常见别名 | 说明 |
| --- | --- | --- |
| Warehouse | 仓库 | 仓储空间的总实体 |
| Zone | 库区 / 区域 | 仓库内的功能分区 |
| Location | 库位 / 储位 / Bin | 库存实际存放位置 |
| SKU | 物料 / 商品 / 库存单位 | 最小管理粒度的商品 |
| Inventory | 库存 / 库存明细 / 库存余额 | 需要在系统中统一口径 |
| Inbound Order | 入库单 / 收货单前置单据 | 业务请求，不等于收货结果 |
| Outbound Order | 出库单 / 发货单前置单据 | 业务请求，不等于发运结果 |
| Receipt | 收货单 / 收货结果 | 入库执行结果 |
| Shipment | 发运单 / 发货结果 | 出库执行结果 |
| Carrier | 承运商 / 物流商 / 运输商 | 负责运输执行的主体 |
| Putaway | 上架 | 将库存放入目标库位 |
| Picking | 拣货 / 取货 | 按订单从库位取货 |
| Packing | 包装 / 打包 | 拣货后整理包装 |
| Loading | 装车 | 货物装上运输工具 |
| Replenishment | 补货 / 补位 | 将库存补到拣货位置 |
| Cycle Count | 盘点 | 周期性核对账实一致性 |

## 使用原则

- 对外文档优先使用主术语
- 系统字段可保留别名，但要写清对应关系
- 当别名有歧义时，以本体定义为准
- 不要让别名替代正式概念定义
